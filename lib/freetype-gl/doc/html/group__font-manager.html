Font manager

Synopsis

Structure in charge of caching fonts.

Example Usage:

#include "font-manager.h"
int main( int arrgc, char *argv[] )
{
font_manager_t * manager = manager_new( 512, 512, 1 );
texture_font_t * font = font_manager_get( manager, "Mono", 12, 0, 0 );
return 0;
}
