vec4 Union Reference

Detailed Description

Tuple of 4 floats.

Each field can be addressed using several aliases:

  • First component: x, left, r or red
  • Second component: y, top, g or green
  • Third component: z, width, b or blue
  • Fourth component: w, height, a or alpha
Examples
ansi.c, atb-agg.c, benchmark.c, cartoon.c, console.c, distance-field.c, font.c, gamma.c, markup.c, outline.c, and subpixel.c.
