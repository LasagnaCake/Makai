ivec4 Union Reference

Detailed Description

Tuple of 4 ints.

Each field can be addressed using several aliases:

  • First component: x, r, red or vstart
  • Second component: y, g, green or vcount
  • Third component: z, b, blue, width or istart
  • Fourth component: w, a, alpha, height or icount
Examples
atb-agg.c.
