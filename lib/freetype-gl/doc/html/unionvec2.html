vec2 Union Reference

Detailed Description

Tuple of 2 floats.

Each field can be addressed using several aliases:

  • First component: x or s
  • Second component: y or t
Examples
ansi.c, atb-agg.c, benchmark.c, cartoon.c, console.c, font.c, gamma.c, markup.c, outline.c, and subpixel.c.
