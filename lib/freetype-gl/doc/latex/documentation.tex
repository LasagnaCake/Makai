\hypertarget{documentation_introduction}{}\doxysection{Introduction}\label{documentation_introduction}
Freetype GL is a small library for displaying unicode text using a (single) vertex buffer and a single texture where necessary glyphs are tighly packed.

{\bfseries{Example Usage}}\+:

 \hypertarget{documentation_API}{}\doxysection{Application Programming Interface}\label{documentation_API}
The code is fairly simple and organized as follow\+:\hypertarget{documentation_mandatory}{}\doxysubsection{Mandatory}\label{documentation_mandatory}
These are the minimum set of structures and files that are required to load a font and pack glyphs into a texture.


\begin{DoxyItemize}
\item vector~\newline
 This structure loosely mimics the std\+::vector class from c++.
\item texture-\/atlas~\newline
 Texture atlas is used to pack several small regions into a single texture.
\item texture-\/font~\newline
 The texture-\/font structure is in charge of creating bitmap glyphs and to upload them to a texture atlas.
\end{DoxyItemize}\hypertarget{documentation_optional}{}\doxysubsection{Optional}\label{documentation_optional}
These are convenient stuctures and files that facilitate text rendering.


\begin{DoxyItemize}
\item shader~\newline
 Shader loading/compiling/linking.
\item vertex-\/buffer~\newline
 Generic vertex buffer structure inspired by pyglet (python).
\item markup~\newline
 Simple structure that describes text properties.
\item font-\/manager~\newline
 Structure in charge of caching fonts.
\item text-\/buffer~\newline
 Convenient structure for manipulating and rendering text. 
\end{DoxyItemize}
